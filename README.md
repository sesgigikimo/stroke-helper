# 小學日記筆順輔助工具

輸入一句話，逐字練習筆順的小工具。給小朋友寫日記時查筆順用的。

## 線上使用

👉 https://sesgigikimo.github.io/stroke-helper/

瀏覽器打開就能用，不用安裝。

## 功能

- 輸入句子自動拆字，逐字顯示筆順動畫
- 田字格 + 十字/對角輔助線
- 筆順測驗模式（用滑鼠或手指寫）
- 動畫速度調整
- 筆順分解圖（逐筆累加顯示）
- 顯示拼音、注音、部首、筆劃數、字義

## 技術

單一 HTML 檔，無後端，使用：

- [Hanzi Writer](https://hanziwriter.org) — 筆順動畫
- [pinyin-pro](https://github.com/zh-lx/pinyin-pro) — 拼音轉換
- [萌典 API](https://www.moedict.tw/) — 字義、注音、部首
